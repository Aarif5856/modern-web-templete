ModernWeb — Usage Guide

A quick visual walkthrough for customizing and deploying the template.

Overview

All content is configured in config.json and rendered dynamically. You can toggle dark mode and switch color presets directly in the navbar. For local edits, serve the folder with a static server (or the template will fall back to inline JSON).

Dark Mode & Color Presets

Use the navbar toggle for dark mode and the dropdown for Blue/Purple/Teal presets. Values are stored in localStorage so your choice persists across reloads.

Build for Production

Requires Node 18+.

cd modernweb
npm install
npm run build

This outputs optimized assets to dist/ and creates modernweb-v1.0.zip for marketplaces.

Deploy

  • Netlify: Use netlify.toml (publish: dist)
  • Vercel: Use vercel.json or import as static site
  • GitHub Pages: Upload the contents of dist/

Tip: Use Ctrl + F in index.html to locate containers (#features-grid, #pricing-grid, #testimonials-grid) if you want to hard-code markup instead of JSON.
